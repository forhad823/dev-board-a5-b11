Question-3: What is event delegation in the DOM?

In the case of multiple child's event of a Parent element, Instead of adding event listeners to multiple child elements, i can attach one event listener to a parent and detect which child fired the event using event.target.(id or className). it is called event delegation

Question-4: What is event bubbling in the DOM?

Event bubbling means when an event happens on a child element, it also fires the event on its parent(s) if there are any event attached previously in parent element. to stop event bubbling, event.stopPropagation() or event.stopImmediatePropagation() are used as per needed.
